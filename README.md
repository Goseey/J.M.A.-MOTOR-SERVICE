# J.M.A. Motor Service — Website

Premium Next.js 14 landing page for J.M.A. Motor Service, Dublin.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Deploy — zero config needed

## Sections
- **Hero** — Cinematic hero with AI-generated car image + particle canvas
- **Stats** — 4 key stats bar
- **Services** — 8 service cards in bento grid
- **Showcase** — Scroll-driven pinned feature section (avatr.com style)
- **Why Us** — Split layout with Google review
- **FAQ** — Animated accordion
- **Contact** — Form + Google Maps
